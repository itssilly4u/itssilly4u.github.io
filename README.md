## ⛏️ Featured Project: MineCalc (Advanced Fleet Builder)

**Live Link:** [itssilly.xyz/minecalc](https://minecalc.itssilly.xyz/)

MineCalc has evolved from a simple loadout calculator into a comprehensive, dynamic **Fleet Builder**. Whether you are flying solo in a Prospector or coordinating a massive multi-crew operation, this tool lets you mix, match, and calculate your exact fleet output.

**Core Features:**
* **Dynamic Fleet Engine:** Build multi-ship setups with automatic laser size and module slot restrictions.
* **Import, Export & Share:** Save individual ships or full fleets as `.json` files to easily share with your org.
* **True-to-Game Math:** Accurately mimics Star Citizen's additive and multiplicative stat stacking.
* **Rock Cracker Analysis:** Input rock mass and gadgets to see if your fleet can break it safely.
* **Customisation:** Rename ships and enjoy a fully responsive mobile UI.

## 🪨 Featured Project: Rock Reader

**Live Link:** [itssilly.xyz/rockreader](https://itssilly.xyz/rockreader)

Rock Reader is a lightweight, fast utility tool designed for Star Citizen miners. It allows users to input a Remote Cluster Signature and instantly calculates the perfect matches for ore compositions and node cluster sizes.

**Core Features:**
* **Instant Calculation:** Lightning-fast, client-side math with zero page reloads.
* **Up-to-Date Values:** Tuned for the latest Star Citizen Alpha patch.
* **Clean UI:** Dark-themed and responsive; perfect for a second monitor or mobile phone.

## ⚡ Technical Philosophy: Built for Speed
All the tools in this repository are built with a focus on **simple UI, fast execution, and highly efficient web coding.** There are no bloated frameworks or heavy libraries. Everything is written in raw, Vanilla JavaScript, HTML, and CSS. This means the math executes instantly directly in your browser, ensuring zero latency whether you are on a high-end gaming PC or checking stats on your phone.
