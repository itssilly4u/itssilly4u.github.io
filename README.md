## 🪨 Featured Project: Rock Reader 

**Live Link:** [itssilly.xyz/rockreader](https://itssilly.xyz/rockreader)

Rock Reader is a lightweight, fast utility tool designed for Star Citizen miners. It allows users to input a Remote Cluster Signature and instantly calculates the perfect matches for ore compositions (like Quantainium, Bexalite, Laranite, etc.) and node cluster sizes. 

### Features:
* **Instant Calculation:** No page reloads; uses vanilla JavaScript to parse signature math on the fly.
* **Up-to-Date Values:** Tuned for Star Citizen Alpha (currently supports 4.7 values).
* **Clean UI:** Dark-themed, responsive design that looks great on a second monitor or mobile phone while playing.

## ⛏️ Featured Project: MineCalc (Mining Loadout Calculator)

**Live Link:** [itssilly.xyz/minecalc](https://itssilly.xyz/minecalc)

MineCalc is a comprehensive, multi-crew mining loadout calculator. Whether you are flying solo in a Prospector or coordinating a fully-crewed Argo MOLE, this tool lets you mix and match lasers, modules, and gadgets to see your exact fleet output.

### Features:
* **Multi-Crew Support:** Calculate the combined mining power for up to 3 operators simultaneously.
* **True-to-Game Math:** Accurately separates stats into Additive pools (Raw Power, Extraction Rates) and Multiplicative stacks (Inert Filtering, Rock Resistance, Instability) exactly how Star Citizen calculates them.
* **Global Gadgets:** Apply a single mining gadget to your target rock to instantly see the fleet-wide debuffs and buffs.
* **Smart Logic:** Automatically detects laser sizes (S1 vs S2) to apply hidden base stats, categorizes equipment alphabetically, and factors in hard-caps like the 50% Optimal Charge Window limit.
* **Dynamic UI:** Features a sleek, dark-themed interface with color-coded stats (green for good, red for bad) and interactive tooltips.

### Features:
* This entire repository is built to be lightweight and fast:
* **HTML5** & **CSS3** (Custom styling, zero heavy frameworks)
* **Vanilla JavaScript** (For the MineCalc math logic)
